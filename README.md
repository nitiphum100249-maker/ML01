ส่วนที่ 1: การนำเข้าไลบรารีที่จำเป็น (Importing Libraries)

ในขั้นตอนนี้ เราจะทำการนำเข้าเครื่องมือต่างๆ (Libraries) ที่จำเป็นสำหรับการวิเคราะห์ข้อมูล

pandas (pd): ไลบรารีหลักสำหรับการจัดการและวิเคราะห์ข้อมูลในรูปแบบตาราง (DataFrame) ช่วยให้อ่านและจัดการข้อมูลได้ง่ายขึ้น

numpy (np): ไลบรารีสำหรับการคำนวณทางคณิตศาสตร์ขั้นสูง และการจัดการข้อมูลแบบอาร์เรย์ (Array)

matplotlib.pyplot (plt): โมดูลสำหรับการสร้างกราฟและพล็อตข้อมูลภาพ (Data Visualization) ขั้นพื้นฐาน

seaborn (sns): เครื่องมือสร้างกราฟทางสถิติที่ต่อยอดจาก matplotlib ช่วยให้กราฟออกมาสวยงามและวิเคราะห์ความสัมพันธ์ได้ง่ายขึ้น

ส่วนที่ 2: การโหลดและสำรวจข้อมูลเบื้องต้น (Data Loading & EDA)
ขั้นตอนนี้คือการนำเข้าข้อมูลและตรวจสอบโครงสร้างพื้นฐานเพื่อทำความเข้าใจลักษณะของชุดข้อมูล

1. โหลดข้อมูล
df = pd.read_csv('lab2_machine_learning_dataset.csv')

2. ตรวจสอบขนาดของข้อมูล
print("Shape of dataset:", df.shape)

3. ตรวจสอบประเภทของตัวแปร
print("\n--- Data Types ---")
print(df.dtypes)

4. ดูค่าสถิติเชิงพรรณนา
print("\n--- Summary Statistics ---")
print(df.describe(include='all'))

5. ตรวจสอบข้อมูลสูญหาย (Missing Values)
print("\n--- Missing Values ---")
print(df.isnull().sum())

6. ตรวจสอบข้อมูลซ้ำซ้อน (Duplicate Records)
print("\n--- Duplicate Records ---")
print("Total duplicates:", df.duplicated().sum())

7. ดูการกระจายตัวของคลาสเป้าหมาย
print("\n--- Class Distribution (Is_Promoted) ---")
print(df['Is_Promoted'].value_counts())
pd.read_csv(...): เป็นคำสั่งสำหรับอ่านไฟล์ข้อมูล CSV แล้วนำข้อมูลมาเก็บไว้ในตัวแปรตารางที่ชื่อว่า df (DataFrame)

df.shape: ตรวจสอบและแสดง "ขนาดของชุดข้อมูล" ว่ามีจำนวนทั้งหมดกี่แถว (Row) และกี่คอลัมน์ (Column)

df.dtypes: ตรวจสอบ "ชนิดของข้อมูล" ในแต่ละคอลัมน์ เช่น เป็นตัวเลขจำนวนเต็ม (int), ทศนิยม (float), หรือข้อความ (object)

df.describe(include='all'): สรุปค่าสถิติพื้นฐานทั้งหมด เช่น จำนวน (count), ค่าเฉลี่ย (mean), ส่วนเบี่ยงเบนมาตรฐาน (std), ค่าต่ำสุด/สูงสุด รวมถึงข้อมูลที่เป็นข้อความ (Categorical data)

df.isnull().sum(): ตรวจสอบว่ามี "ค่าว่าง" (Missing Values / NaN) อยู่หรือไม่ โดยนับจำนวนรวมของค่าว่างแยกตามแต่ละคอลัมน์

df.duplicated().sum(): ตรวจสอบและนับจำนวน "แถวที่บันทึกข้อมูลซ้ำซ้อนกันทุกคอลัมน์"

df['Is_Promoted'].value_counts(): นับจำนวนและความถี่ของคอลัมน์ Is_Promoted (คลาสเป้าหมาย) เพื่อดูว่ามีพนักงานที่ได้รับการเลื่อนขั้น และไม่ได้เลื่อนขั้น จำนวนอย่างละกี่คน ซึ่งจะช่วยให้เราเห็นความสมดุลของข้อมูล (Class Balance)
